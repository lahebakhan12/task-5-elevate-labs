# Task 5: Exploratory Data Analysis (EDA) – Titanic Dataset

## Description
This task focuses on performing Exploratory Data Analysis (EDA) on the Titanic dataset using Python. The objective is to understand the data structure, identify patterns and relationships, visualize insights, and summarize findings using statistical and graphical techniques.

## Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

## Dataset
- Titanic Dataset (Seaborn built-in)
- 891 records, 15 features

## Work Done
- Data loading and inspection
- Handling missing values
- Univariate and bivariate analysis
- Visualizations (histograms, bar plots, box plots)
- Observations and summary of findings

## Key Insights
- Females had higher survival rates than males
- First-class passengers had better survival chances
- Higher fares generally correlated with survival
- Children showed higher survival probability
